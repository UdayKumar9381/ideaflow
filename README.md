## 🛠️ Technology Stack

### **Frontend (The 3DVERSE)**
- **Framework**: React 18+ (Vite)
- **3D Engine**: @react-three/fiber & @react-three/drei (Three.js)
- **Styling**: Tailwind CSS v4 (Custom Design Tokens)
- **Animations**: Framer Motion (Ambient Physics)
- **State/API**: Axios (Interceptors) + Context API

### **Backend (The Engine)**
- **Framework**: FastAPI (Python 3.10+)
- **Database**: MySQL with **SQLAlchemy (Async)**
- **Migrations**: Alembic
- **Security**: JWT-based Authentication + Passlib (Bcrypt) hashing

## 🚀 Quick Start

### 1. Initialize Engine (Backend)
```bash
cd backend
pip install -r requirements.txt
# Configure .env
uvicorn app.main:app --reload
```

### 2. Launch 3DVERSE (Frontend)
```bash
cd frontend
npm install
npm run dev
```
